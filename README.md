This is a single interactive file with 5 tabs covering everything you need end-to-end:
📅 Daily Plan — a checkbox checklist for all 20 days (4–23 July), mapped exactly to your exam dates, with specific tasks each day across grammar, vocab, and listening/speaking/writing practice
📐 Grammar — the 7 grammar points that cover most of what A2 tests (verb conjugation, word order, want/omdat, niet/geen, de/het, modal verbs, prepositions)
🔤 Vocabulary — flip-flashcards across 7 topic sets (personal info, family, daily routine, housing, health, work/school, time)
💬 Phrases — survival phrases for speaking situations (greetings, clarification, requests, opinions, apologies, phone calls) plus writing connectors
🖥️ Quiz — a mixed CBT-style quiz pulling from all of the above, one question at a time with instant feedback
Note: checkboxes and quiz progress reset on page reload — treat it as your daily working tool rather than a saved tracker. Start with the Daily Plan tab today, and work through Grammar/Vocab/Phrases as each day calls for them.
